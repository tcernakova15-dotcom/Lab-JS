# Учёт личных финансов

> Лабораторная работа №5 — Работа с DOM-деревом и событиями в JavaScript

---

## Описание

Веб-приложение для учёта личных финансов. Позволяет добавлять и удалять транзакции, просматривать историю и следить за общим балансом в реальном времени.

---

## Структура проекта

```
/
├── index.html          # Точка входа, разметка приложения
├── style.css           # Стили
└── src/
    ├── index.js        # Главный модуль: инициализация, обработчики событий
    ├── transactions.js # Модуль данных: хранение и управление транзакциями
    ├── ui.js           # Модуль DOM: отрисовка, обновление интерфейса
    └── utils.js        # Утилиты: генерация ID, форматирование дат и сумм
```

## Запуск

Так как проект использует ES-модули (`type="module"`), файлы нельзя открыть напрямую через `file://`. Нужен локальный HTTP-сервер.

**Через VS Code (рекомендуется):**
установите расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer), затем нажмите `Go Live` в статус-баре.

**Через Node.js:**
```bash
npx serve .
```

**Через Python:**
```bash
python -m http.server 8080
```

Затем откройте `http://localhost:8080` в браузере.

## Модули

### `utils.js`
Вспомогательные функции, не зависящие от DOM или бизнес-логики.

| Функция | Описание |
|---|---|
| `generateId()` | Генерирует уникальный строковый ID |
| `formatDate(date)` | Форматирует `Date` в строку `ДД.ММ.ГГГГ ЧЧ:ММ` |
| `truncateWords(text, n)` | Возвращает первые `n` слов строки (по умолчанию 4) |
| `formatAmount(amount)` | Форматирует число как `+1 500.00 ₽` или `−200.00 ₽` |

### `transactions.js`
Управление данными. Хранит массив транзакций и предоставляет CRUD-операции.

| Функция | Описание |
|---|---|
| `addTransaction(data)` | Создаёт объект транзакции, добавляет в массив, возвращает его |
| `removeTransaction(id)` | Удаляет транзакцию по ID, возвращает `boolean` |
| `getTransactions()` | Возвращает копию массива всех транзакций |
| `calculateTotal()` | Считает сумму всех транзакций |
| `findById(id)` | Находит транзакцию по ID |

### `ui.js`
Работа с DOM. Ничего не знает о бизнес-логике — только рисует и обновляет.

| Функция | Описание |
|---|---|
| `renderTransaction(t)` | Добавляет строку транзакции в таблицу |
| `removeTransactionRow(id)` | Удаляет строку из таблицы, показывает заглушку если пусто |
| `updateTotal(total)` | Обновляет баланс в шапке |
| `showTransactionDetail(t)` | Показывает полные детали транзакции |
| `clearTransactionDetail()` | Сбрасывает блок деталей до заглушки |
| `showFieldError(fieldId, msg)` | Показывает ошибку под полем формы |
| `clearFormErrors()` | Очищает все ошибки формы |

### `index.js`
Точка входа. Импортирует все модули и настраивает обработчики событий:
- `submit` на форме — валидация и добавление транзакции
- `click` на `<table>` — делегирование: удаление или просмотр деталей

## Ответы на контрольные вопросы

**1. Как получить доступ к элементу на веб-странице с помощью JavaScript?**

Через методы объекта `document`:
```js
document.getElementById('my-id');          // по id
document.querySelector('.my-class');       // по CSS-селектору (первый)
document.querySelectorAll('tr[data-id]');  // по CSS-селектору (все)
```

**2. Что такое делегирование событий?**

Делегирование — паттерн, при котором обработчик навешивается не на каждый дочерний элемент, а на их общего родителя. Событие всплывает (`bubbling`) от источника вверх по дереву, и родитель определяет цель через `e.target`. В проекте это используется для кнопок удаления и строк таблицы:
```js
table.addEventListener('click', (e) => {
  const deleteBtn = e.target.closest('[data-delete-id]');
  if (deleteBtn) { /* удалить */ }

  const row = e.target.closest('tr[data-id]');
  if (row) { /* показать детали */ }
});
```
Преимущества: меньше обработчиков в памяти, работает для динамически добавляемых элементов.

**3. Как изменить содержимое элемента DOM после его выборки?**

```js
const el = document.getElementById('total-display');

el.textContent = 'Новый текст';       // только текст, безопасно
el.innerHTML = '<b>HTML</b>-контент'; // может содержать теги
el.setAttribute('class', 'negative'); // изменить атрибут
el.style.color = 'red';               // изменить стиль напрямую
```

**4. Как добавить новый элемент в DOM-дерево?**

```js
// 1. Создать элемент
const tr = document.createElement('tr');
const td = document.createElement('td');
td.textContent = 'Данные';

// 2. Собрать структуру
tr.appendChild(td);

// 3. Вставить в дерево
document.querySelector('tbody').appendChild(tr);    // в конец
document.querySelector('tbody').prepend(tr);        // в начало
parent.insertBefore(tr, referenceNode);             // перед узлом
```
