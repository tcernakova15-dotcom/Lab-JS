# 🧮 Калькулятор — Индивидуальный проект по JavaScript

## Описание проекта

Веб-калькулятор с современным интерфейсом, реализованный на чистом **Vanilla JS** (ES6+ модули) без сторонних фреймворков. Поддерживает базовые арифметические операции, историю вычислений, ввод с клавиатуры и адаптивный дизайн.

## Авторы

- **[Ваше Имя Фамилия]** — студент группы [XX]

---

## Основные функции

| Функция | Описание |
|---|---|
| Арифметика | Сложение, вычитание, умножение, деление |
| Десятичные числа | Поддержка дробей через «,» |
| Смена знака | Кнопка `+/−` меняет знак операнда |
| Процент | `%` вычисляет процент от предыдущего значения |
| Backspace | Удаление последнего введённого символа |
| История | Сохраняет до 20 последних вычислений (LocalStorage) |
| Очистка истории | Кнопка `HIST` очищает всю историю |
| Клавиатура | Полная поддержка ввода с клавиатуры |
| Валидация | Деление на ноль и некорректный ввод — с понятным сообщением об ошибке |

## Структура проекта

```
calculator/
├── index.html          # Разметка приложения
├── css/
│   └── style.css       # Стили (CSS-переменные, адаптивность, анимации)
├── js/
│   ├── main.js         # Точка входа, обработчики событий
│   ├── calculator.js   # Модуль логики и состояния
│   ├── history.js      # Модуль истории вычислений
│   └── ui.js           # Модуль рендеринга DOM
└── README.md
```

### Модульная архитектура

- **`calculator.js`** — вся логика вычислений и состояние (без DOM).
- **`history.js`** — хранение и управление историей (LocalStorage).
- **`ui.js`** — рендеринг: обновление дисплея, истории, подсветка операторов.
- **`main.js`** — точка входа: связывает модули, обрабатывает клики и клавиатуру.

## Инструкция по запуску

### Способ 1 — Live Server (рекомендуется)

1. Откройте папку проекта в **VS Code**.
2. Установите расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
3. Нажмите **Go Live** в нижней панели.

> Необходимо запускать через HTTP-сервер, так как проект использует ES-модули (`type="module"`), которые не работают напрямую через `file://`.

### Способ 2 — Python HTTP-сервер

```bash
cd calculator
python -m http.server 8080
# Откройте http://localhost:8080 в браузере
```

### Способ 3 — Node.js (`serve`)

```bash
npm install -g serve
serve calculator
```

**Зависимости:** браузер с поддержкой ES2020+ (Chrome 80+, Firefox 75+, Safari 14+). Никаких npm-зависимостей нет.

## Управление с клавиатуры

| Клавиша | Действие |
|---|---|
| `0–9` | Ввод цифр |
| `.` или `,` | Десятичная точка |
| `+` `-` `*` `/` | Операторы |
| `Enter` или `=` | Вычислить |
| `Backspace` | Удалить символ |
| `Escape` | Очистить историю |
| `%` | Процент |

## Примеры использования

### Обычное вычисление

Введите: 125 + 75 =

50 × 2 − 30 = 70

```
Введите: 5 ÷ 0 =

```

## Технические решения

- **ES6-модули** (`import`/`export`) — разделение кода по ответственности.
- **Делегирование событий** — один обработчик на всю панель кнопок.
- **CSS-переменные** — единая тема, легко настраивается.
- **LocalStorage** — история сохраняется между сессиями браузера.
- **ARIA-атрибуты** — базовая доступность для скринридеров.
- **Анимации** — рябь при клике, плавный слайд при загрузке (CSS-only).
